# OnNotes (React + Firebase)

This repository contains the OnNotes frontend built with React + Firebase.

The old Laravel application has been removed.

## Run (development)

```bash
cd frontend
pnpm install
pnpm run dev --host 127.0.0.1 --port 5173
```

Open the URL printed by Vite (e.g. `http://127.0.0.1:5173`).
